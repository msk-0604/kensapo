## システム設定ファイル

- 全体DB設定: `supabase/setup.sql`
- 既存環境への通知追加: `supabase/RUN_PUSH_NOTIFICATIONS.sql`
- Storage設定は `setup.sql` に含まれる（単独の `storage.sql` は参考用）
- 本番環境変数: Vercel Project Settingsで管理
- 本番コード: GitHubのmainブランチ
