# KenSapo 取扱説明書

建設現場の日報・写真・予定をスマホで記録するアプリです。  
**60代の方でも使えるよう、文字とボタンを大きくしています。**

---

## 1. はじめに

| 項目 | 内容 |
|------|------|
| アプリ名 | **KenSapo（ケンサポ）** |
| 本番URL | https://kensapo.vercel.app |
| 対応端末 | スマホ（iPhone・Android）のブラウザ |
| ログイン | メールアドレス ＋ パスワード |

### 画面の下にある4つのボタン

| ボタン | 意味 |
|--------|------|
| **ホーム** | 今日やることを選ぶ |
| **予定** | 今日の作業予定・開始・終了 |
| **現場** | 工事現場の一覧・詳細 |
| **設定** | 自分の名前・ログアウトなど |

---

## 2. ログイン方法

1. https://kensapo.vercel.app を開く
2. **ログイン** タブを選ぶ
3. メールアドレスとパスワードを入力
4. **ログイン** ボタンを押す

### 初めて使う人（会社の管理者）

1. **新規登録** タブを選ぶ
2. 会社名・お名前・メール・パスワードを入力
3. **アカウント作成** を押す
4. すでに登録済みのメールの場合は、**ログイン** タブから入る

### 顔認証・指紋について

- ログイン画面には **出ていません**（シンプルにするため）
- 使いたい人は、ログイン後に **設定** → **顔認証・指紋の設定** から登録できます

---

## 3. 毎日の使い方（作業員向け）

### いちばんよく使う流れ

```
ホーム → 今日の予定を見る → 作業を開始する → （作業） → 作業を終了する
```

#### 手順

1. **ホーム** を開く
2. **今日の予定を見る** を押す
3. 自分の予定を確認
4. **作業を開始する** を押す（作業が始まったとき）
5. 作業が終わったら **作業を終了する** を押す

### 写真を撮るとき

1. **ホーム** → **写真を撮る**
2. 現場の名前を選ぶ
3. 写真を選んでアップロード

### ログアウトするとき

1. **設定** を開く
2. 一番下の **ログアウトする** を押す

---

## 4. 現場監督・事務向けの使い方

### 現場を登録する

1. **現場** → **新しい現場を登録する**
2. 現場名・住所・担当者などを入力
3. **この現場を登録する** を押す

### 作業員を登録する

1. **設定** → **作業員を管理する**
2. **作業員を登録する**
3. 名前・電話・職種を入力して保存

### 予定を入れる

1. **予定** を開く
2. 日付を選ぶ（**今日** ボタンで今日に戻れる）
3. **新しい予定を追加** から、現場・作業員・時間を入力

### 日報を書く（日報）

1. **現場** → 現場を選ぶ → **日報**
2. **新しい日報を書く**
3. 天気・作業内容などを入力して保存
4. **日報を作成** で文章を自動生成
5. 必要なら編集し、**印刷・PDF** で出力

### 工事進行チェック

1. **現場** → 現場を選ぶ → **工事進行**
2. 終わった項目にチェックを入れる

---

## 5. 困ったとき

| 症状 | 対処 |
|------|------|
| ログインできない | パスワードを確認。新規登録済みならログインタブを使う |
| 画面がずっと読み込み中 | 電波を確認。アプリを閉じて開き直す |
| 現場がない | 監督・管理者に現場登録を依頼する |
| 予定がない | 監督に予定登録を依頼する |
| 写真が上がらない | 写真のサイズを小さくする。電波のよい場所で再試行 |
| ログアウトしたい | **設定** の一番下 |

### 管理者向け：最初のセットアップ

データベースが未設定のとき、黄色い案内が出ます。

1. **設定手順を見る**（または `/setup`）を開く
2. Supabase で SQL を1回実行
3. アプリを開き直す

## 6. 用語の説明

| 言葉 | 意味 |
|------|------|
| 現場 | 工事の場所（マンション新築、リフォーム現場など） |
| 予定 | 誰が・いつ・どの現場で働くかの予定表 |
| 日報 | その日の作業内容の記録 |
| 日報 | 現場の作業内容の記録 |
| 工事進行 | 工事の進み具合のチェックリスト |

## 7. お客様への見せ方（デモ）

1. ログイン
2. **ホーム** で大きな3ボタンを見せる
3. **予定** で「作業を開始する」「作業を終了する」を実演
4. **写真を撮る** で1枚アップロード
5. 日報 → 日報（設定済みの場合）
