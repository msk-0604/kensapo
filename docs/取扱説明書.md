# KenSapo 取扱説明書

KenSapoは、建設現場の予定・作業員・写真・日報・工事進行をスマートフォンで管理するシステムです。

本番URL: https://kensapo.vercel.app

## 1. ログイン

1. 本番URLを開きます。
2. メールアドレスとパスワードを入力します。
3. 「ログイン」を押します。

初めて会社を登録する管理者は「新規登録」から会社名・氏名・メールアドレス・パスワードを登録します。

## 2. 画面下のメニュー

- **ホーム**: 今日使う操作を選びます。
- **予定**: 作業予定の確認・登録・開始・終了を行います。
- **現場**: 現場情報・写真・日報・工事進行を管理します。
- **設定**: 作業員・通知・顔認証・ログアウトを管理します。

## 3. 毎日の基本操作

### 作業を開始・終了する

1. 「予定」を開きます。
2. 自分の予定を確認します。
3. 作業開始時に「作業を開始する」を押します。
4. 作業終了時に「作業を終了する」を押します。

### 写真を追加する

1. 「ホーム」から「写真を撮る」を押します。
2. 現場を選びます。
3. 撮影日・タイトル・メモを入力します。
4. 写真を撮影または選択して保存します。

### 日報を書く

1. 「現場」から対象現場を選びます。
2. 「日報」を開きます。
3. 「新しい日報を書く」を押します。
4. 日付・天気・人数・作業内容などを入力します。
5. 「日報を保存する」を押します。

## 4. 管理者・現場監督の操作

### 現場を登録する

1. 「現場」を開きます。
2. 「新しい現場を登録する」を押します。
3. 現場名・住所・担当者・期間などを入力して保存します。

### 作業員を登録する

1. 「設定」を開きます。
2. 「作業員を管理する」を押します。
3. 作業員の名前・電話番号・職種を登録します。

### 予定を登録する

1. 「予定」を開きます。
2. 「新しい予定を追加する」を押します。
3. 日付・現場・作業員・時間・作業内容を入力して保存します。

### 工事進行を更新する

1. 「現場」から対象現場を選びます。
2. 「工事進行」を開きます。
3. 完了した項目を押してチェックします。

## 5. 通知をオンにする

通知をオンにすると、同じ会社の別ユーザーが予定・現場・日報・写真・工事進行を更新した際に、ロック画面へ通知が届きます。

1. 「設定」を開きます。
2. 「通知をオンにする」を押します。
3. 端末の確認画面で「許可」を選びます。

### iPhoneの場合

Safariの共有ボタンから「ホーム画面に追加」を行い、ホーム画面のKenSapoから通知をオンにしてください。

通知は設定をオンにした端末にだけ届きます。自分自身の操作通知は自分には送信されません。

## 6. 顔認証・指紋認証

対応端末では「設定」の「顔認証・指紋の設定」から登録できます。

ログイン画面ではメールアドレスとパスワードを使います。顔認証・指紋は、対応端末と設定が有効な場合に補助として利用できます。

## 7. 困ったとき

- **ログインできない**: メールアドレスとパスワードを確認し、登録済みの場合は「ログイン」タブを使用します。
- **画面が開かない**: 電波状況を確認し、一度閉じて開き直します。
- **写真が保存できない**: 電波のよい場所で再度試し、写真サイズが大きすぎないか確認します。
- **通知が届かない**: 端末設定でKenSapoの通知が許可されているか確認します。
- **予定や現場が見つからない**: 同じ会社のアカウントでログインしているか管理者へ確認します。

## 8. ログアウト

「設定」の最下部にある「ログアウトする」を押します。
