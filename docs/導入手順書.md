# KenSapo 導入手順書

## 1. 導入前に確認する情報

お客様から次を確認します。

- 会社名
- 管理者の氏名
- 管理者のメールアドレス
- 最初に登録する現場名
- 作業員の氏名・電話番号・職種
- 利用するスマートフォンの台数
- 利用開始日

## 2. 管理者アカウントを作る

1. https://kensapo.vercel.app を開きます。
2. 「新規登録」を選びます。
3. 会社名・管理者名・メールアドレス・パスワードを入力します。
4. 登録後、ログインできることを確認します。

パスワードは8文字以上とし、お客様自身に管理してもらいます。

## 3. 初期データを登録する

1. 「現場」で最初の現場を登録します。
2. 「設定」→「作業員を管理する」で作業員を登録します。
3. 「予定」で翌営業日または当日の予定を登録します。
4. 対象現場に写真を1枚登録します。
5. 対象現場に日報を1件登録します。

## 4. スマートフォンへ追加する

### iPhone

1. SafariでKenSapoを開きます。
2. 共有ボタンを押します。
3. 「ホーム画面に追加」を押します。
4. ホーム画面のKenSapoからログインします。

### Android

1. ChromeでKenSapoを開きます。
2. 右上メニューを押します。
3. 「ホーム画面に追加」または「アプリをインストール」を押します。

## 5. 通知を設定する

事前にSupabase SQL Editorで `supabase/RUN_PUSH_NOTIFICATIONS.sql` を実行します。

各端末で次を行います。

1. KenSapoの「設定」を開きます。
2. 「通知をオンにする」を押します。
3. 端末の通知確認で「許可」を選びます。

## 6. 通知テスト

2人以上のアカウントを使います。

1. 受信側端末で通知をオンにして、端末をロックします。
2. 操作側アカウントで予定を追加または作業を開始します。
3. 受信側のロック画面に通知が表示されることを確認します。
4. 通知を押して関連画面が開くことを確認します。

## 7. 操作説明

次の順に実演すると約20分で説明できます。

1. ログイン
2. 予定の確認
3. 作業開始・終了
4. 現場の登録
5. 写真の追加
6. 日報の保存
7. 工事進行チェック
8. 通知設定
9. ログアウト

## 8. 導入完了条件

以下を満たした時点で導入完了とします。

- 管理者がログインできる
- 現場と作業員が1件以上登録されている
- 予定を登録・確認できる
- 写真と日報を保存できる
- 利用端末で通知を受信できる
- 管理者へ取扱説明書を渡している
